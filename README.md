## What is WatchWise?

WatchWise is a modern web application that helps you discover, track, and share movies tailored to your taste. It combines advanced AI-powered recommendations, collaborative group suggestions, and a beautiful, intuitive interface.

**Key Features:**
- Personalized movie recommendations based on your preferences and watch history
- Group movie suggestions for friends and couples, with real-time compromise logic
- OAuth login (Google, GitHub) and privacy-first onboarding
- Customizable taste profile and daily mood questionnaire
- Watchlists, favorites, and history tracking
- Responsive, mobile-friendly UI

---

## Project Structure

- `watchwise-backend/` — Fastify + MongoDB backend, API, recommendation logic
- `watchwise-frontend/` — Next.js (App Router) frontend, React components, UI/UX
